Proyecto ITSE

Software académico para la administración del centro de información del Instituto Tecnológico Superior de Ébano. Incluye funcionalidades para la gestión de préstamos, inventario y registro de usuarios. El sistema fue desarrollado en Java con conexión a base de datos MySQL, permitiendo su uso como una herramienta completa en bibliotecas institucionales.
